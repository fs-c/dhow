Dhow is a JSX-powered static site generator for Node.js

It turns a .js file like this

import Dhow from 'dhow'

const Home = () => (
    <main>
        <h3>This is my home</h3>
        <p>On the internet obviously</p>
    </main>
)

export const Head = () => (
    <>
        <title>Home page</title>
    </>
)

export default Home

Into this

<!DOCTYPE html>
<html>
    <head>
        <title>Home page</title>
    </head>
    <body>
        <div class="dhow">
            <main>
                <h3>This is my home</h3>
                <p>On the internet obviously</p>
            </main>
        </div>
    </body>
</html>
